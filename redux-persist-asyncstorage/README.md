## 🚀 How to use

- Bootstrap this project:
  > `npx create-react-native-app -t https://github.com/amandeepmittal/react-native-examples/tree/master/redux-persist-asyncstorage`
- Start the project:
  - `yarn ios` -- open on iOS
  - `yarn android` -- open on Android
